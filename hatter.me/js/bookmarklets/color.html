Color Bookmarklets

These bookmarklets let you play with colors on any web page that does not make heavy use of images. They require Mozilla 1.5a+ because they use CSS3 HSL (Hue-Saturation-Lightness) colors.

(To keep bookmarklets in order to use them on other web pages, drag them to your Bookmarks Toolbar. Or, install them all at once.)

Bookmarklet Description Works in
+hue [H = H + 1/24] Red becomes green, etc. Details
-hue [H = H - 1/24] Green becomes red, etc. Details
red [H = 0] Details
green [H = 1/3] Details
blue [H = 2/3] Details
+light [L = L ^ (5/6)] Details
-light [L = L ^ (6/5)] Details
invert lightness [L = 1 - L] Dark green becomes light green, white becomes black. Details
midbright [L = 0.5] Details
+sat [S = S + 0.15] Details
-sat [S = S - 0.15] Details
no sat [S = 0] (grayscale) Details
full sat [S = 1] Details

Browsers: is Internet Explorer for Windows, is Netscape 7 or Mozilla, is Netscape 4, and is Opera.
