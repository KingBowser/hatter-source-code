Search bookmarklets

Search bookmarklets.

(To keep bookmarklets in order to use them on other web pages, drag them to your Bookmarks Toolbar. Or, install them all at once.)

Bookmarklet Description Works in
google Searches Google for the selected text. Details
google site search Searches Google for pages on the site containing selected text. Details
google site search: all Searches Google for all pages on the site. Details
google site search: title Searches Google for pages on the site with the same title. Details
num=100 Changes a Google search to show 100 results per page. Details
num=10 Changes a Google search to show 10 results per page. Details
num=1 Changes a Google search to show 1 result per page. Details
filter=0 Changes a Google search to remove the limit of 2 URLs per site. Details
@google Takes you from any search to the same search on Google. Details
@alltheweb Takes you from any search to the same search on Alltheweb. Details
@teoma Takes you from any search to the same search on Teoma. Details
@msn Takes you from any search to the same search on MSN Search. Details
@altavista Takes you from any search to the same search on Altavista. Details
wayback newest Goes to the newest archived copy of a page. Details
wayback search Searches the Wayback Machine for all old copies of the page. Details

Browsers: is Internet Explorer for Windows, is Netscape 7 or Mozilla, is Netscape 4, and is Opera.
