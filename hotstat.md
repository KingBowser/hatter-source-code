# Introduction #

HotStat, Display the diff of `jstat -J-Djstat.showUnsupported=true -snap [pid]`

# Details #

```
wget https://hatter-source-code.googlecode.com/svn/trunk/hotstat/hotstatall.jar
```

Usage:
```
Usage:
  java -jar hotstatall.jar [options] [<PID> [<interval> [<count>]]]
```
